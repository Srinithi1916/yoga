# Jeevanam 360

Premium wellness website built with a React + Vite frontend and a Spring Boot + MongoDB backend.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Spring Boot
- MongoDB
- Razorpay

## Local Run

Frontend:

```powershell
npm install
npm run dev
```

Backend:

```powershell
npm run backend:dev
```

## Build

```powershell
npm run build
npm run backend:build
```
